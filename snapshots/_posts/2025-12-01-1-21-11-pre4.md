This pre-release adds dispenser functionality and fixes more bugs.

## Nautilus

Dismounting a nautilus or zombie nautilus while on land will place the player at the same location dismounting a pig would.

## Zombie Nautilus

Zombie nautili now drop additional rotten flesh when killed with looting:

| Count       | 0    | 1    | 2    | 3    | 4    | 5    | 6    | Average |
| ----------- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ------- |
| No Looting  | 1/4  | 1/4  | 1/4  | 1/4  |      |      |      | 1.5     |
| Looting I   | 1/8  | 2/8  | 2/8  | 2/8  | 1/8  |      |      | 2       |
| Looting II  | 1/12 | 2/12 | 3/12 | 3/12 | 2/12 | 1/12 |      | 2.5     |
| Looting III | 1/16 | 2/16 | 3/16 | 4/16 | 3/16 | 2/16 | 1/16 | 3       |
