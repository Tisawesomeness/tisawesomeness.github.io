This snapshot continues to remodel baby mobs while technical changes happen behind the scenes.

## Mobs

The hitbox sizes for baby mobs have changed:

| Mob        | Width        | Height          | Eye Height       |
| ---------- | ------------ | --------------- | ---------------- |
| Baby Camel | 0.765 → 1.02 | 1.06875 → 1.425 | 1.02375 → 1.365  |
| Baby Fox   | 0.3 → 0.36   | 0.35 → 0.42     | 0.2 → 0.24       |
| Baby Goat  | 0.45 → 0.495 | 0.65 → 0.715    | 0.5525 → 0.60775 |

Foxes now crouch for **25 ticks** before pouncing, up from 15 ticks.
