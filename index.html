Minecord

Minecord is a lightweight yet powerful Discord bot. It has useful commands for you Discord server such as server lookup, name history, a recipe and item browser, and much more. It's open source too!

Invite the bot at https://minecord.github.io.

Minecraft Reverse Engineering

Every time Mojang releases a new snapshot, I peek inside the source code to figure out how new features work. I then share my findings with people in the technical Minecraft community and on the wiki.

Check out the findings here!

Plugins

  • BetterPreview - Processes Minecraft chat previews client-side to improve privacy, stability, and performance
  • PortalSuppressor - Prevents mobs from spawning in portals
  • DiamondNuggets - Adds a diamond nugget crafting recipe
  • NoToasts - Disables warning toasts when needed
