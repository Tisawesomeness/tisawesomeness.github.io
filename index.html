About Me

My username is Tis_awesomeness, but you can call me Tis. I'm a software engineer from North Carolina. You may see me occasionally in the technical Minecraft and Discord developer communities.

Projects

Minecord

A lightweight yet powerful Discord bot with Minecraft recipe browsing, server status, profile lookup, and more utilities.

Snapshot Reverse Engineering

An in-depth analysis of the source code for every new Minecraft snapshot, uncovering details not found in the changelogs.

Almaminer

Developer/sysadmin for a Minecraft server bringing students and professionals together after the pandemic. Custom work includes a revamped end fight, the Block Hunt minigame, several custom plugins, and contributions to ViaVersion and BreweryX.

NameHistorian

A plugin that records and displays name changes over time for players who join your server.

CookieJar

A Fabric mod allowing developers and server administrators to view and edit cross-server cookies.

ChunkDeputy

A plugin that pre-generates chunks when players are offline to improve server performance.

DiamondNuggets

A plugin adding craftable "diamond nuggets" that players can trade.
